
# ポケモンバトルシミュレーター

## 概要
PokeAPIを使ったポケモン検索・バトルシミュレーションアプリです。

## 機能
-ポケモン名で検索してステータスを確認
-好きなポケモンを２体選んでバトルシミュレーション
-バトルログで戦闘の経過を確認

##　使用技術
-React/Vite
-PokeAPI（外部API）
-コンポーネント分割設計(App/SearchBar/PokemonCard/BattleArena)
